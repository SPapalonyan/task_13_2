# Извлечение ключевых слов из отзывов

## Описание задачи

В рамках данного задания необходимо реализовать функцию `find_keywords()`, которая обрабатывает текстовые отзывы и извлекает из них ключевые слова с оценкой важности.

## Входные данные

CSV-файл с отзывами, содержащий столбец:

- `normalized_text` — предобработанный текст (токенизация и лемматизация уже выполнены)

## Функциональность

Функция `find_keywords(file_path)` выполняет следующие действия:

1. Загружает CSV-файл в pandas DataFrame
2. Для каждого текста из столбца `normalized_text`:
   - извлекает ключевые слова
   - рассчитывает важность каждого слова
3. Сохраняет результат в новый столбец:
   - `key_words` — список кортежей вида `(слово, score)`
4. Возвращает обновлённый DataFrame

## Используемый подход

Для извлечения ключевых слов используется метод TF-IDF (Term Frequency – Inverse Document Frequency).

Данный метод позволяет:
- выделять наиболее значимые слова в тексте
- уменьшать влияние часто встречающихся слов
- получать числовую оценку важности каждого слова

## Используемые библиотеки
